# bookbot

BookBot is a simple command-line tool that analyzes a text file and prints statistics about it, including word count and character frequency.

## How to use

Run the program with a path to a text file:

```
python main.py path/to/your/book.txt
```

The program will output a report showing the total word count and how many times each letter appears in the text.

## Requirements

Python 3.8 or newer
